Node.js のインストール

Node.js のインストール

Node.js 公式サイト から LTS 版パッケージ（2019年1月時点で 10.15.0）をダウンロードしてインストールしてください。

複数のバージョンを管理したい場合は、Nodist(Windows) や nodebrew(Mac) を使うのがおすすめです。
